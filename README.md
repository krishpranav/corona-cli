# corona-cli
Track covid 19 cases from your cli 

[![forthebadge](https://forthebadge.com/images/badges/made-with-crayons.svg)](https://forthebadge.com)

# Installation
```
git clone https://github.com/krishpranav/corona-cli
cd corona-cli
yarn install
yarn dev
```

# Data can be fetched
```
Country
Cases
Deaths
Recovered
Active
Critical
Per Million
```
